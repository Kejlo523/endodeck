# EndoDeck enclosure

The STL files fit the current EndoDeck phone builds:

- Huawei P8 Lite ALE-L21: `EndoDeck-P8Lite-Front-Bezel.stl`, `EndoDeck-P8Lite-Rear-Stand.stl`
- Samsung Galaxy A6 2018 SM-A600FN: `EndoDeck-SamsungA6-Front-Bezel.stl`, `EndoDeck-SamsungA6-Rear-Stand.stl`

Print the front bezel and rear stand as separate parts, then use the preview renders to confirm orientation before slicing.

These source assets stay versioned. Release binaries are generated into `dist/` and `release/` by CI.
